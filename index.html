Abstract


   Arranging class schedules is a facilitator for all institutions to complete their academic work according to planned goals. The planning of class schedules is therefore consistent with teachers, students, classrooms, times, and subjects in the curriculum. But class scheduling still uses the same method, which is class scheduling using humans to arrange the data each time. This causes problems in scheduling complicated classes. This must be resolved according to several constraints, such as the time each student and teacher are free. Number of classrooms Classroom capacity Number of students registered in each subject The needs of teachers who want to teach at certain times, etc., when making corrections The class scheduling process must therefore be repeated until the information is suitable. causing more delays.
  To make class scheduling more efficient Both in terms of resource use, costs and time spent organizing class schedules. and the results of scheduling classes that are consistent with the limitations and responding to the needs of students and teachers Therefore, it is necessary to choose appropriate and effective methods to solve this problem. Moreover, the method used must handle the constraints related to the problem well. Various application methods have been used to solve this problem. Each method has its own advantages and disadvantages. As a result, it is not possible to directly compare those methods to see which is the best way to solve the university scheduling problem. Class scheduling is a complex process, such as shuffling time slots for students who have failed a course in the previous semester. Scheduling of instructors serving courses outside the field of study Finding a process for arranging class schedules to ensure suitability in the subjects. Instructor Classroom Curriculum Number of Students The class scheduling problem is classified as a time scheduling problem, which is an NP-complete problem. That is, in the processing The time required to solve a problem varies significantly with the size of the problem to such an extent that it may not be possible to find an answer in a reasonable amount of time. Moreover, class scheduling problems may not be solved under the conditions of the problem itself.
